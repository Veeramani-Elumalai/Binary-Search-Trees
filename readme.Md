# Binary Search Tree (JavaScript)

A simple Binary Search Tree (BST) implementation in JavaScript built as part of a data structures learning project.  
This project demonstrates tree creation, traversal, insertion, deletion, balancing, and rebalancing.

---

## Features

- Build a balanced Binary Search Tree from an array
- Insert and delete values
- Find a value in the tree
- Tree traversals:
  - Level Order
  - Inorder
  - Preorder
  - Postorder
- Check if the tree is balanced
- Rebalance an unbalanced tree
- Visualize the tree structure using `prettyPrint`

---
